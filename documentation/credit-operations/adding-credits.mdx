The `add_credit(request: AddCreditRequest)` method allows you to add credits to your account. You can specify the amount and other relevant details in the request.

### Example

```python
import asyncio
from mira_sdk import MiraClient, AddCreditRequest

async def add_credits():
    # Initialize the client
    client = MiraClient(
        base_url="https://api.mira.example.com",
        api_token="your-api-token"
    )

    # Create a request object
    request = AddCreditRequest(amount=50, description="Adding credits for additional usage")

    # Add credits
    response = await client.add_credit(request)
    print("Credits Added:", response)

if __name__ == "__main__":
    asyncio.run(add_credits())
```

### Output

The output confirms the addition of credits:

```plaintext
Credits Added: {"new_balance": 150}
```
