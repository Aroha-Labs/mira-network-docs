## Generating Text with a Model

The `generate(request: AiRequest)` method allows you to generate text using a specific model. This involves creating an `AiRequest` object, specifying the model and input messages.

### Example

```python
import asyncio
from mira_sdk import MiraClient, Message, AiRequest

async def generate_text():
    # Initialize the client
    client = MiraClient(
        base_url="https://api.mira.example.com",
        api_token="your-api-token"
    )

    # Create a request object
    request = AiRequest(
        model="mira/llama3.1",
        messages=[
            Message(role="system", content="You are a helpful assistant."),
            Message(role="user", content="Can you summarize the latest news?")
        ],
        model_provider=None
    )

    # Generate and print the response
    response = await client.generate(request)
    print("Generated text:", response)

if __name__ == "__main__":
    asyncio.run(generate_text())
```

### Output

The output will be the text generated by the specified model:

```plaintext
Generated text: "The latest news highlights advancements in AI technology..."
```
