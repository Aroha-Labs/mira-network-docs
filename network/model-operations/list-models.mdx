# Model Operations

The Mira SDK provides intuitive methods for interacting with AI models hosted on the Mira Network. This section explains how to use the SDK to list available models and generate text using a specified model.

## Listing Available Models

The `list_models()` method retrieves a list of all models available on the Mira Network. This can be useful for exploring the models you can utilize in your application.

### Example

```python
import asyncio
from mira_sdk import MiraClient

async def list_available_models():
    # Initialize the client
    client = MiraClient(
        base_url="https://api.mira.example.com",
        api_token="your-api-token"
    )

    # Fetch and print available models
    models = await client.list_models()
    print("Available models:", models)

if __name__ == "__main__":
    asyncio.run(list_available_models())
```

### Output

The output will be a list of model identifiers available for use:

```plaintext
Available models: ['mira/llama3.1', 'mira/babbage', 'mira/davinci']
```
