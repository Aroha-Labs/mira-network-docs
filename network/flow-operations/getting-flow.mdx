<Note>Since the testnet is on Beta, the flow operations can be unstable.</Note>

The `get_flow(flow_id: str)` method fetches details of a specific flow.

### Example

```python
import asyncio
from mira_sdk import MiraClient

async def get_flow_details():
    # Initialize the client
    client = MiraClient(
        base_url="https://api.mira.example.com",
        api_token="your-api-token"
    )

    # Retrieve details of a specific flow
    flow_id = "flow1"
    flow_details = await client.get_flow(flow_id)
    print("Flow details:", flow_details)

if __name__ == "__main__":
    asyncio.run(get_flow_details())
```

### Output

```plaintext
Flow details: {'id': 'flow1', 'name': 'Example Flow', 'description': 'An example flow for demonstration.'}
```
